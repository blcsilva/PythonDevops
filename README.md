# README: Scripts Python para DevOps

## Introdução

Este repositório contém uma coleção de 10 scripts Python projetados para facilitar a vida de desenvolvedores e profissionais de DevOps. Esses scripts abordam tarefas comuns de automação, monitoramento e gerenciamento que são essenciais para manter a eficiência e a confiabilidade em ambientes de desenvolvimento e produção.

## Objetivo

O objetivo destes scripts é fornecer ferramentas úteis para automatizar e otimizar processos do dia a dia em DevOps. Com a crescente complexidade das infraestruturas e a necessidade de agilidade, automação é uma necessidade.

## Scripts Incluídos

1. **Automação de Backup**  
   Realiza backups automáticos de arquivos e diretórios importantes, garantindo a integridade dos dados.

2. **Monitoramento de Uso de Disco**  
   Monitora o uso do disco e alerta se a capacidade utilizada ultrapassar um limite configurável.

3. **Gerenciamento de Logs**  
   Realiza a rotação de logs antigos, mantendo a organização e prevenindo o uso excessivo de espaço em disco.

4. **Implantação de Aplicações**  
   Automatiza o processo de implantação de uma aplicação em um servidor remoto via SSH.

5. **Verificação de Saúde de Serviços**  
   Verifica se serviços essenciais estão em execução, ajudando na manutenção da infraestrutura.

6. **Coleta de Métricas de Sistema**  
   Coleta e exibe métricas de uso da CPU e da memória, permitindo monitoramento de desempenho em tempo real.

7. **Gerenciamento de Dependências**  
   Atualiza automaticamente as dependências de um projeto Python, mantendo o ambiente de desenvolvimento atualizado.

8. **Notificação por Email**  
   Envia notificações por e-mail em caso de falhas ou eventos críticos, permitindo resposta rápida a problemas.

9. **Limpeza de Arquivos Temporários**  
   Remove arquivos temporários de diretórios específicos, ajudando a manter o sistema limpo e organizado.

10. **Implementação de Pipeline CI/CD Simples**  
    Define uma sequência básica de tarefas em uma pipeline, servindo como base para integrações contínuas.

## Como Usar

1. **Instalação do Python**: Certifique-se de que o Python esteja instalado em sua máquina. Você pode baixá-lo em [python.org](https://www.python.org/downloads/).

2. **Configuração do Ambiente**: Para scripts que requerem bibliotecas adicionais (como `paramiko` ou `psutil`), instale-as usando `pip`:
   ```bash
   pip install paramiko psutil
   ```

3. **Execução dos Scripts**: Navegue até o diretório onde os scripts estão salvos e execute-os com o seguinte comando:
   ```bash
   python nome_do_script.py
   ```

4. **Personalização**: Sinta-se à vontade para modificar os scripts de acordo com suas necessidades específicas.
